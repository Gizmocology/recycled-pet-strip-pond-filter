# Recycled PET Strip Pond Filter

Sistem filtrasi kolam ikan berbasis keberlanjutan (sustainability) dengan Up-Flow Vortex memanfaatkan 100% strip botol plastik (PET) daur ulang sebagai media filter mekanis dan biologis, dan sistem backwash terintegrasi.

- **Precision Media:** Pembagian ukuran strip PET untuk filtrasi bertahap.
- **Integrated Backwash:** Memudahkan pembersihan tanpa harus membongkar media.
- **Venturi Aeration:** Menambah suplai oksigen secara pasif ke dalam air.

## ğŸ›  Spesifikasi Teknis Media
Media filter disusun dengan rasio volume **1:2:2** berdasarkan ketebalan strip:
1. **Lapisan Bawah (Kasar):** Strip PET 5mm (Rasio 1) - Menangkap kotoran besar.
2. **Lapisan Tengah (Sedang):** Strip PET 3mm (Rasio 2) - Filtrasi mekanis menengah.
3. **Lapisan Atas (Halus):** Strip PET 1mm (Rasio 2) - Filtrasi halus dan luas permukaan untuk bakteri pengurai.
