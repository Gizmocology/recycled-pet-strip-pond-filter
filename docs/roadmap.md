# Proyek Roadmap: Recycled PET Strip Pond Filter

Dokumen ini berisi rencana pengembangan proyek dari tahap ide hingga implementasi.

## Fase 1: Riset & Prototipe Alat (Sedang Berjalan)
- [ ] Riset desain alat pemotong botol (Bottle Cutter) yang mampu menghasilkan strip 1mm secara konsisten.
- [ ] Pengumpulan bahan baku (Botol PET bekas air mineral).
- [ ] Eksperimen pemotongan untuk menentukan rasio tarikan dan kelenturan strip.
- [ ] Finalisasi desain tabung filter Vortex.

## Fase 2: Produksi Media & Perakitan
- [ ] Produksi massal strip PET (ukuran 5mm, 3mm, dan 1mm) hingga memenuhi volume tabung.
- [ ] Perakitan jalur pipa inlet (Vortex), outlet (Venturi), dan jalur backwash.
- [ ] Instalasi media ke dalam tabung berdasarkan lapisan rasio 1:2:2.
- [ ] Uji kebocoran dan kelancaran aliran air (Flow test).

## Fase 3: Pengujian & Validasi Data
- [ ] Running test selama 30 hari pada kolam ikan.
- [ ] Pencatatan data harian (kejernihan, perilaku ikan, debit air) di `data/test-log.md`.
- [ ] Evaluasi efektivitas sistem backwash (seberapa mudah endapan dibuang).
- [ ] Analisis kejenuhan media filter (kapan media 1mm mulai tersumbat).

## Fase 4: Digitalisasi & Smart Monitoring (Masa Depan)
- [ ] Integrasi sensor kekeruhan air (Turbidity Sensor) menggunakan mikro kontroler.
- [ ] Pengembangan notifikasi ke smartphone jika filter sudah terlalu kotor dan butuh backwash.
- [ ] Monitoring debit air secara real-time untuk mendeteksi penurunan performa pompa.

## Fase 5: Publikasi & Kontribusi
- [ ] Dokumentasi foto dan video untuk setiap komponen.
- [ ] Mengunggah video tutorial pembuatan dan hasil uji coba ke channel YouTube **gizmotech**.
- [ ] Mengubah status repositori GitHub dari **Private** menjadi **Public**.
- [ ] Menyusun file `LICENSE` agar proyek bisa dikembangkan bersama oleh komunitas dan bermanfaat untuk masyarakat umum.
