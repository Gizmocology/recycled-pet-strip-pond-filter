# Panduan Pembuatan Strip PET

Bagian paling krusial dari proyek ini adalah konsistensi ukuran media filter. Berikut adalah tahapannya.

## 1. Seleksi Bahan Baku
- Gunakan botol plastik PET bening (bekas air mineral).
- Pastikan botol dalam keadaan silinder (permukaan rata), bukan yang memiliki banyak lekukan/profile agar hasil potongan strip stabil.
- Jika permukaan botol berlekuk/berprofil perlu diratakan dulu dengan cara dipompa dan dipanaskan sambil botol diputar.

## 2. Alat Pemotong (Bottle Cutter)
*Draf: detail alat pemotong strip (metode cutter dan bearing).*
- **Setting 5mm:** Digunakan untuk lapisan dasar.
- **Setting 3mm:** Digunakan untuk lapisan tengah.
- **Setting 1mm:** Digunakan untuk lapisan atas (membutuhkan ketelitian ekstra).

## 3. Teknik Pemotongan
1. Lepaskan label dan bersihkan sisa lem (lem bisa menyumbat pori-pori bakteri).
2. Potong bagian bawah botol secara manual sebagai titik awal.
3. Masukkan ke alat pemotong dan tarik strip dengan kecepatan konstan.
4. Kelompokkan atau jadikan satu strip sesuai ukurannya.
